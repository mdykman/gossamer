Welcome to Gossamer
Gossamer is a completely new approach to MVC web app development.
mvC - Controller
In the current environment, the language you choose to program with largely determines the framework you use. Furthermore, once a particular langauge is selected, it is difficult to employ another language which might be better suited to a particular processs of locagical case.
Gossamer supports many programing languages. Any language which has an implementation supporting the JSR-223 Scripting Engine Interface is a candidate for use under the Gossamer framework. Languages currently supported by Gossamer include:
  • groovy
  • javascript
  • python (jython)
  • ruby
  • beanshell
  • judo
  • jelly
  • scheme
  • PHP
    • Script Console The default credentials are: admin/gossamer
